# Chatbot PDF RAG Azure

Projeto de portfólio: chatbot interativo baseado em conteúdo de PDFs utilizando RAG (Retrieval-Augmented Generation) e Azure OpenAI.

## Estrutura do projeto

- `inputs/` → PDFs para teste
- `src/` → código Python
- `notebooks/` → notebooks experimentais
- `outputs/` → arquivos gerados pelo projeto

## Tecnologias

- Python 3.10
- OpenAI + Azure OpenAI via API
- LangChain
- PyPDF2
- FAISS (armazenamento vetorial)
- NumPy e Pandas

## Objetivo

Criar um chatbot capaz de responder perguntas com base em PDFs, usando embeddings e RAG.
